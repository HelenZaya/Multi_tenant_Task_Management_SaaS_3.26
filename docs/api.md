# API Documentation

Base path: `/api/v1`

## Core endpoints
- `POST /auth/register`
- `POST /auth/login`
- `POST /auth/refresh`
- `GET /auth/me`
- `POST /auth/invite`
- `GET /workspaces`
- `POST /workspaces`
- `GET /boards`
- `GET /boards/summary`
- `GET /boards/:boardId`
- `POST /boards`
- `POST /cards`
- `PATCH /cards/:cardId/move`
- `GET /users`
- `GET /reports`
- `POST /reports/generate`

Interactive Swagger UI is served at `https://localhost/docs`.
